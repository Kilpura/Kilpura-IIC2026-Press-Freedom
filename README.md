# Visualización de Datos: Libertad de Prensa / Nivel de Ingresos

Este repositorio contiene el preprocesamiento de datos, el análisis exploratorio y la generación de gráficos estáticos para un proyecto de Visualización de la Información (InfoVis) desarrollado para el curso **IIC2026**. 

El producto final de este análisis fue implementado como una página web interactiva alojada en **Glitch**, diseñada para permitir a los usuarios explorar y sacar sus propias conclusiones sobre cómo el poder económico de un país se relaciona con su libertad de prensa entre los años 2019 y 2024.

Para lograrlo, se procesaron datos sociodemográficos y económicos, generando mapas coropléticos que ilustran de forma clara la distribución geográfica de estas variables.

## Estructura del Repositorio

*   **`IIC2026/` (Directorio de Trabajo)**
*   **`2025/Clases/`**:
    *   Material teórico de apoyo sobre buenas prácticas de diseño de visualizaciones, uso de formas, colores y principios de InfoVis.

## Stack Tecnológico

**Procesamiento de Datos y Gráficos Estáticos (Backend):**
*   **Python:** `pandas` (limpieza y cálculos), `numpy` (operaciones numéricas).
*   **Librerías de Visualización:** `geopandas` (manejo de shapefiles) y `matplotlib` (generación de figuras y mapas de calor).

**Despliegue Web (Frontend):**
*   **Glitch:** Plataforma de alojamiento (HTML / CSS / JavaScript) donde reside la visualización interactiva final.
